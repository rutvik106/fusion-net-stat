# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately before disclosing it publicly.

**Email**: security@yourdomain.com
- Include "FusionNet Stat Security" in the subject
- Provide detailed description of the vulnerability
- Include steps to reproduce if possible

**What to expect**:
- We'll respond within 48 hours
- We'll provide a timeline for fixing the issue
- We'll notify you when the fix is released
- We'll credit you in the release notes (optional)

Please do NOT open a public issue for security vulnerabilities.
